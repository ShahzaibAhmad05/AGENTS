# AGENTS

Config files for guiding my Cursor/VsCode/AntiGravity agents. I configure them for giving extremely short, human-tone outputs with maximum token saving. Refer to `AGENTS.md` for more details.

This has been saving me ~70% in token consumption and 60% of my brain cells since I don't have to move back-and-forth to make the agent do the right thing. Responses have become 80% more polite and bear-able in pressured situations. The numbers are based on my own observations though.

Would recommend trying!
